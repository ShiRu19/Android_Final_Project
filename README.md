# School Epidemic Report System

## Introduce

一款疫情通報系統，可以讓學校組織內的確診者使用這款系統進行通報，且管理者可以確實知道確診者，做出立即的應對措施。



主要分成行政端與學生端，行政端可以審核學生的確診足跡，並且回朔先前的課表來製造確診足跡資料。

學生端可以進行登入，供給行政端進行審核，方便整合確診足跡。
